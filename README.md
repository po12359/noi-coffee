# noi coffee Website

삿포로 소재 프리미엄 카페 **noi coffee**의 공식 웹사이트입니다.

## 🏗️ 프로젝트 특징
- **프리미엄 디자인**: 차분하고 고급스러운 카페 분위기를 재현한 Glassmorphism 및 부드러운 애니메이션 적용.
- **반응형 웹**: 모바일, 태블릿, 데스크탑 모든 환경에서 최적화된 레이아웃 제공.
- **성능 최적화**: Vite를 사용한 빠른 로딩과 효율적인 자산 관리.
- **인터랙티브 UI**: 스크롤 기반 페이드인 효과 및 직관적인 네비게이션.

## 🛠️ 기술 스택
- **Core**: HTML5, Vanilla JavaScript
- **Styling**: Vanilla CSS (Custom Design System)
- **Build Tool**: Vite
- **Deployment Ready**: GitHub Pages 등에 즉시 배포 가능

## 🚀 시작하기

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 개발 서버 실행:
   ```bash
   npm run dev
   ```

3. 프로덕션 빌드:
   ```bash
   npm run build
   ```

## 📸 주요 섹션
- **Hero**: 카페의 아이덴티티를 보여주는 강렬한 시각적 섹션.
- **About**: noi coffee의 철학과 스토리를 담은 소개.
- **Menu**: 시그니처 메뉴 하이라이트.
- **Gallery**: 제공된 고화질 사진들을 활용한 갤러리.
- **Location**: 방문을 유도하는 위치 정보.
